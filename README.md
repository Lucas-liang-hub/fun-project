# 流萤 · 自习室

沉浸式专注计时器，配有动态场景动画和 AI 陪伴。

**在线预览：** [https://lucas-liang-hub.github.io/fun-project/](https://lucas-liang-hub.github.io/fun-project/)

---

## 功能特性

- **🍅 番茄钟专注计时** — 25 分钟专注倒计时，支持暂停/继续
- **🌌 6 种动态场景** — 樱花、星空、霜花、萤火虫、深海、秋叶，每个场景都有独特的粒子动画和色彩氛围
- **🤖 AI 学习伴侣** — 内置聊天机器人，在你专注时随机发送鼓励话语
- **📊 专注统计** — 记录每次专注时长，统计今日和累计专注时间
- **📜 专注证书** — 每次完成专注后会生成专属证书，可保存分享
- **🌊 流动进度** — 专注时间越长，场景动画动态增强，带来沉浸式体验
- **🔐 用户系统** — 支持注册/登录，数据存储在本地 localStorage
- **🎨 精美 UI** — 毛玻璃效果、渐变色彩、柔和动画，营造安静专注氛围

## 技术栈

| 技术 | 用途 |
|------|------|
| HTML5 Canvas | 所有场景动画粒子系统 |
| 原生 JavaScript | 计时器逻辑、用户系统、AI 对话 |
| CSS3 | 毛玻璃设计、渐变、动画 |
| localStorage | 用户数据、专注记录持久化 |

## 快速使用

1. 打开 [在线预览](https://lucas-liang-hub.github.io/fun-project/)
2. **注册/登录** — 创建你的自习账号，或直接点击中央计时器开始（游客模式）
3. **选择场景** — 左侧场景选择器或开始前的场景预览
4. **开始专注** — 点击中央的圆形计时器开始 25 分钟专注
5. **AI 陪伴** — 点击右下角按钮打开聊天，随时获取鼓励
