# 流萤 · 自习室

沉浸式专注计时器，配有动态场景动画和 AI 陪伴。

**在线预览：** [https://lucas-liang-hub.github.io/fun-project/](https://lucas-liang-hub.github.io/fun-project/)

![项目截图](/screenshot.png)

---

## 功能特性

- **🍅 番茄钟专注计时** — 25 分钟专注倒计时，支持暂停、继续、重置
- **🌌 6 种动态场景** — 樱花、星空、霜花、萤火虫、深海、秋叶，每个场景都有独特的 Canvas 粒子动画和色彩氛围
- **🤖 AI 学习伴侣** — 内置聊天机器人，专注时随机发送鼓励话语，支持自然语言对话
- **📊 专注统计** — 记录每次专注时长，统计今日和累计专注时间
- **📜 专注证书** — 每次完成专注生成专属证书，含场景名称、专注时长、励志语录，可保存为文本文件
- **🌊 流动进度** — 专注时间越长，场景动画动态增强（粒子加速、光效叠加），带来沉浸式反馈
- **🔐 用户系统** — 支持注册/登录，数据存储在本地 localStorage
- **🎨 视觉设计** — 深色主题 + 毛玻璃效果 + 金色点缀配色，营造安静、沉静的专注氛围

## 场景一览

自习室内置 6 种动态场景，每种场景都通过 Canvas 2D 实时渲染，风格各异：

| 场景 | 图标 | 氛围 | 粒子/元素 | 配色 |
|------|------|------|-----------|------|
| 樱花 | 🌸 | 春日宁静 | 飘落的花瓣、光晕 | 粉色、金色 |
| 星空 | ⭐ | 银河深邃 | 旋转星群、流星轨迹 | 深蓝、银白 |
| 霜花 | ❄️ | 冬日清冷 | 冰晶生长、雪花飘落 | 冰蓝、白色 |
| 萤火虫 | ✨ | 夏夜梦幻 | 飞行光点、草丛光晕 | 黄绿、暖金 |
| 深海 | 🌊 | 海洋深邃 | 鱼群游弋、气泡、光影波纹 | 深蓝、青色 |
| 秋叶 | 🍂 | 秋日温暖 | 落叶飘零、阳光光束 | 橙红、赭石 |

专注时间越长，场景中的"流动"参数值越高，动画节奏加快、特效叠加增强，形成正向反馈。

## 技术栈

| 技术 | 用途 |
|------|------|
| **HTML5 Canvas 2D** | 全部 6 种场景的粒子系统和动画渲染（~1300 行渲染代码） |
| **原生 JavaScript** | 计时器逻辑、用户系统、AI 对话引擎、UI 交互 |
| **CSS3** | 毛玻璃 (backdrop-filter)、渐变色、Flex/Grid 布局、响应式适配 |
| **localStorage** | 用户账号、专注记录持久化存储 |
| **Google Fonts** | Noto Serif SC / Noto Sans SC 中文字体 |
| **GitHub Pages** | 静态托管与在线预览 |

## 快速使用

1. 打开 [在线预览](https://lucas-liang-hub.github.io/fun-project/)
2. **注册/登录** — 创建你的自习账号，或直接点击中央计时器以游客模式使用
3. **选择场景** — 点击左侧场景选择器（桌面端）或开始前的场景预览卡片
4. **开始专注** — 点击中央的圆形计时器开始 25 分钟专注倒计时
5. **AI 陪伴** — 点击右下角金色按钮打开聊天窗口，可与 AI 伴侣自由对话
6. **查看统计** — 右侧面板显示今日专注分钟数，点击左下角日志按钮查看历史记录

## 遇到的问题与解决

### 场景渲染性能优化
6 个场景各有 30-80 个粒子对象同时更新，早期直接使用多个 setInterval 导致帧率不稳。改用统一的 requestAnimationFrame 主循环 + deltaTime 驱动后，动画帧率稳定在 60fps。

### 流动进度系统的设计
需要让场景动画随专注时长递增强度（粒子速度、数量、光效），但动画参数变化需要平滑。方案是定义一个 0-1 的 flowProgress 基准值，每个场景的渲染函数在循环中对各自粒子属性（速度、透明度、尺寸）乘以 (0.5 + flow * 0.5)，并根据 flow 阈值叠加额外特效。

### 登录系统与数据持久化
项目无后端，所有数据存储在浏览器 localStorage 中。用户注册时以 JSON 格式存储账号和密码，专注记录按日期归类存储。游客模式下无需登录即可使用，专注数据在刷新后不可恢复，鼓励用户注册。

### 配色与视觉一致性
深色主题下要确保文字可读性、毛玻璃通透感和金色点缀不显得杂乱。通过 CSS 变量统一管理所有颜色（--ink、--gold、--paper、--glass 等），所有 UI 组件引用这些变量，确保全局视觉一致性。

### 响应式布局
桌面端有左侧场景选择器和右侧统计面板，移动端屏幕较窄时自动隐藏这些侧边栏，计时器居中显示，聊天窗口全屏宽度。通过 CSS 媒体查询和 pointer-events 控制交互层叠。
